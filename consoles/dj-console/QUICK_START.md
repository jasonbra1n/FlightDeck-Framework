## üöÄ How to Use

### 1. **Initialize Audio**
- Click the **"Initialize Audio"** button (required for mobile browsers)
- This enables audio playback permissions

### 2. **Load Audio Files**
- Click **"LOAD"** button on either deck (Left or Right)
- Select audio files (MP3, WAV, OGG, AAC supported)
- Or drag and drop files onto the waveform area

### 3. **Start Mixing**
- Press **Play** buttons to start tracks
- Use **Channel Faders** to adjust individual track volumes
- Use **Crossfader** to blend between decks
- Adjust **EQ controls** (HI, MID, LOW) for frequency balance

### 4. **Advanced Features**
- **Hot Cues**: Click numbered buttons to set/recall cue points
- **Pitch Control**: Use pitch fader or +/- buttons for tempo adjustment
- **Sync**: Enable SYNC to match BPM between decks
- **Jog Wheels**: Drag for scratching effects

### Audio Issues

- **Mobile**: Ensure you tap "Initialize Audio" before loading files

- **Size**: Large files may take longer to load and analyze
- **Browser Support**: Modern browsers with Web Audio API support required
